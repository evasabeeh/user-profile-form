# Multi-Step User Profile Form

### Frontend
Vite + React.js + Tailwind CSS

### Backend
Node.js + Express.js + MongoDB

### Key points
1. File upload - Multer
2. Database schema - Mongoose
3. Database - MongoDB
4. API call - Axios
5. Deployment - Render
